Flynet software used in Melis, Siwanowicz, and Dickinson 2023

### Build Dependencies 

```bash
sudo apt install qtbase5-dev libvtk9-dev libarmadillo-dev libboost-python-dev libboost-numpy-dev
```

### Installation
cd into source directory and run
```bash
pip install .
```
